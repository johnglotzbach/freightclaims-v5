# Image Assets

Place your image assets in the appropriate folders:

- **logos/** — Company logos (FreightClaims logo, favicon, OG image, partner logos)
- **leadership/** — Team/leadership headshots for the About Us page
- **marketing/** — Marketing assets, hero images, feature screenshots
- **icons/** — Custom icons, app icons, PWA icons

## Naming Convention

Use lowercase with hyphens: `logo-full-color.png`, `john-smith-headshot.jpg`

## Recommended Formats

- Logos: SVG (preferred) or PNG with transparency
- Photos: JPEG or WebP for compression
- Icons: SVG or PNG at 2x resolution
- Favicon: Place `favicon.ico` in the `public/` root directory
- OG Image: 1200x630px, place as `og-image.jpg` in `public/`
